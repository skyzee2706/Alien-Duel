# 👽 Alien Duel: Premium PvP Wagering on Alien Network

**Alien Duel** is an innovative, high-stakes asynchronous PvP wagering mini-app developed specifically for the **Alien Network**. It allows players to engage in fast-paced 1v1 duels like *Dice Duel* and *Lucky Number* within a secure, high-performance internal economy. By combining a futuristic dark-mode interface with immersive haptic feedback and atomic transaction security, Alien Duel provides a premium, transparent, and tactile gaming experience where users can seamlessly wager, win, and withdraw ALIEN tokens directly to their personal wallets with zero latency and full on-chain verification.
